## What Baymax can do at this belt
ลูกพูดประโยคเดียว "ขาแม่บวมค่ะ" → Baymax กวาดทุกอวัยวะ:

```text
👁 ตา      → เก็บอาการพื้นผิว (ขาบวม)
🗃️ ความจำ  → ดึงประวัติ (CHF, CKD, เบาหวาน, น้ำท่วมปอด)
💊 ยา      → ยาที่แม่กินอยู่
🔁 เคสเก่า → หมอคนก่อนทำอะไร
🧠 สมอง    → ต่อเส้นทางว่าทำไมบวม
🛑 เบรก    → กันให้ยาขับน้ำแรงเกิน
🗣️ ปาก     → พูด 2 ภาษา (หมอ / ลูก)
```

## ยังขาดอะไร (ทำไมยังเป็นแค่สายขาว)
- เจอเคสเก่าที่ "สำเร็จ" แล้วมีแนวโน้มทำซ้ำเดิม
- ยังไม่รู้ว่า **สถานะวันนี้ของแม่เปลี่ยนไปรึยัง**
- Retrieval อย่างเดียว = ของถูกกว่า ใครก็ทำได้

→ เลื่อนขั้นได้ที่ **🟡 Yellow Belt** เมื่อ Baymax เริ่มถามว่า "เคสเก่าสำเร็จจริง แต่วันนี้เหมือนเดิมไหม?"
